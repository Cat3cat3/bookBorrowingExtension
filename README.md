# 豆瓣×四川图书馆

这是一个MVP，欢迎大家和我一起使用。

## 功能

- 在豆瓣的任意图书信息页面，右侧将显示四川省图书馆的馆藏资源信息，方便了解该图书在四川省图书馆是否可借阅。

## 使用方法

1. 在edge浏览器市场搜索并安装扩展程序。（chrome要我$5保护费，我不愿意
2. 打开豆瓣网页。
3. 浏览任意图书信息页面，右侧将显示四川省图书馆的馆藏资源信息。
